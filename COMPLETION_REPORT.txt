⚙️ THREADING MODEL
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    Thread #1: JavaFX Main
    ├─ Renders UI
    ├─ Handles user input
    └─ Updates based on listener callbacks

    Thread #2: OrderService
    └─ Generates orders every 5 seconds

    Thread #3: RouteBuilderService
    └─ Groups orders into batches

    Thread #4: DispatcherService
    └─ Assigns batches to shippers

    Thread #5: ShipperWorker (Alice)
    └─ Delivers Alice's orders

    Thread #6: ShipperWorker (Bob)
    └─ Delivers Bob's orders

    Thread #7: ShipperWorker (Charlie)
    └─ Delivers Charlie's orders

    Thread #8: ShipperWorker (Diana)
    └─ Delivers Diana's orders

    Total: 8 concurrent threads, NO UI BLOCKING ✅
